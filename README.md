# 📝 Two-Per-Week

## 🚀 스터디 목표
- 꾸준한 알고리즘 문제 풀이를 통해 코딩 테스트 실력 향상 및 문제 해결 능력 강화

## 📋 스터디 규칙
- 한 주에 **자율적**으로 2문제를 선택하여 풀이
- 한 주는 매주 **월요일 오전 9시 (09:00)**에 시작하여, **차주 월요일 오전 9시 (09:00)**에 마감됩니다.
- `주차(Week00)` > `개인 이름(gildong)` 폴더 내에 풀이 코드를 push
- 못 풀면 벌금 10,000원

## 💰 페널티
- 마감 시간까지 할당된 문제(2문제)를 모두 풀고 제출하지 못할 경우, **벌금 10,000원**이 부과됩니다.

## 💡 참고 사항
- 풀이 언어는 자유롭게 선택할 수 있습니다.
